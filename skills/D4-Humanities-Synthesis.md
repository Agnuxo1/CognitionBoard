# Humanities Synthesis (D4)

## Role
Combine multiple humanities domains into a unified analysis.

## When to Use
- Task involves ≥2 humanities domains (history, philosophy, psychology, linguistics, geography, economics, politics, law, culture, arts)
- Examples: economic history, philosophy of law, political psychology, cultural linguistics

## Protocol

```
1. Identify all humanities domains involved
2. Load each domain's skill (B1-B6, C1-C2, D2, C6)
3. Find cross-domain narratives and frameworks
4. Integrate methodologies from each domain
5. Produce unified analysis with multi-domain perspective
```

## Cross-Domain Frameworks

| Domain A | Domain B | Bridge Framework |
|----------|----------|-------------------|
| History → Economics | Institutional change → Market evolution | Path dependence |
| Philosophy → Law | Ethics → Jurisprudence | Natural law theory |
| Psychology → Politics | Cognition → Decision-making | Behavioral IR |
| Linguistics → Culture | Discourse → Identity | Linguistic anthropology |
| Geography → History | Space → Time | Annales school |
| Arts → Philosophy | Aesthetics → Epistemology | Critical theory |

## Example

**Task:** "Analyze how economic inequality drives political polarization"
- Path: D5·A1·B6·C1·D4·D6
- Skills: Master → Compress → Economics → Politics → HumSynth → Output
- Integration: Piketty r>g (B6) + populism theory (C1) + comparative cases (B1)
