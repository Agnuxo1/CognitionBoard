# Science Synthesis (D3)

## Role
Combine multiple scientific domains into a unified analysis.

## When to Use
- Task involves ≥2 scientific domains (math, physics, chemistry, biology, CS, engineering, medicine, astronomy)
- Examples: quantum biology, bioinformatics, computational chemistry, astrophysics modeling

## Protocol

```
1. Identify all scientific domains involved
2. Load each domain's skill (A2-A6, C3-C5)
3. Find cross-domain principles and analogies
4. Integrate formalisms from each domain
5. Produce unified analysis with multi-domain notation
```

## Cross-Domain Principles

| Domain A | Domain B | Bridge Principle |
|----------|----------|-----------------|
| Physics → Biology | Thermodynamics → Evolution | Free energy minimization |
| Math → CS | Topology → Networks | Graph invariants |
| Chemistry → CS | Molecular graphs → Data structures | Isomorphism algorithms |
| Astronomy → Physics | Observation → Theory | Model fitting |
| Medicine → Biology | Pathology → Systems biology | Network perturbation |

## Example

**Task:** "Model protein folding using quantum mechanics"
- Path: D5·A1·A3·A5·C3·D3·D6
- Skills: Master → Compress → Physics → Biology → Medicine → SciSynth → Output
- Integration: Quantum Hamiltonian (A3) + AlphaFold architecture (A5) + drug-binding (C3)
